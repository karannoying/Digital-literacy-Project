## Part A – Coding Practice: CodeChef

**Platform used:** CodeChef (codechef.com)

I created an account on CodeChef and completed a beginner-level challenge from the 'Learn to Code' section — a basic input/output problem requiring the user to read two integers and print their sum. While the problem itself is straightforward, the experience of using an online judge for the first time was genuinely instructive.

Submitting code to an automated system that tests it against hidden test cases is a fundamentally different experience from running code locally. The judge is unambiguous — it either passes or it does not — and that precision is something I found both challenging and oddly satisfying. It also introduced me to the concept of standard input/output formatting, which matters even in simple problems.

CodeChef's structured learning path and contest calendar make it easy to set progressive goals, and I can see myself returning to it regularly as I develop stronger programming fundamentals across my first two years.

## Part B – Google Workspace Collaboration: Digital Literacy Quiz

**Platform used:** Google Forms + Google Sheets (forms.google.com)

I built a 5-question *Digital Literacy Awareness Quiz* using Google Forms, designed to be shared with batchmates as a quick awareness check. The form includes a mix of multiple choice and short answer questions covering UPI safety, phishing recognition, and the purpose of professional platforms like GitHub and LinkedIn. Responses are automatically captured in a linked Google Sheet.

Building this form taught me more than I expected. Designing questions that are clear, unambiguous, and appropriately challenging for a general audience requires a level of care that is easy to underestimate. I had to revise two of my questions after realising the wording could lead to multiple valid interpretations — a lesson that applies directly to survey design, requirements documentation, and test planning in engineering and R&D contexts.

The automatic response capture in Google Sheets also demonstrated how integrated cloud tools can eliminate manual data collection entirely, which is directly relevant to collaborative academic projects and lab work.
